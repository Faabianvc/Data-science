# Data Science & Machine Learning Projects

Este repositorio está dedicado al desarrollo y ejecución de proyectos de Ciencia de Datos, abarcando desde la limpieza y exploración de datos (EDA) hasta la implementación de modelos de aprendizaje automático (Machine Learning).

## 💡 Objetivo del Proyecto
El propósito de este espacio es aplicar metodologías analíticas para resolver problemas complejos, utilizando datasets del mundo real para generar insights de valor y modelos predictivos robustos. 

Se trabajarán conceptos fundamentales como:
* **Análisis Exploratorio (EDA):** Visualización de tendencias y correlaciones.
* **Regresión y Clasificación:** Predicción de variables continuas y categóricas.
* **Evaluación de Modelos:** Uso de métricas como R-squared, MSE y matrices de confusión.

---

## 🛠️ Requisitos y Entorno
Para ejecutar los notebooks y scripts de este proyecto, es necesario contar con un entorno de Python 3.x y las siguientes librerías instaladas:

### Extensiones y Librerías Principales
* **Pandas:** Manipulación y análisis de estructuras de datos.
* **NumPy:** Soporte para vectores, matrices y funciones matemáticas de alto nivel.
* **Matplotlib:** Creación de visualizaciones estáticas y gráficas de calidad.
* **Seaborn:** Interfaz de alto nivel para visualizaciones estadísticas atractivas.
* **Scikit-learn:** Herramientas para minería y análisis de datos, incluyendo algoritmos de ML.
* **SciPy:** Utilidades para computación científica y técnica.
